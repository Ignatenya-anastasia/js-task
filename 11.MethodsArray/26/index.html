На входе n – количество элементов массива, затем пользователь заполняет массив с клавиатуры. Вывести true, если все строки в массиве имеют длину более 3 символов. Использовать метод every. Входные: ["apple", "banana", "cherry"] → Результат: true Входные: ["dog", "cat", "apple"] → Результат: false Входные: ["hschool", "company", "cherry"] → Результат: true
